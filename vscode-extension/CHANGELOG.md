# Changelog

All notable changes to the Synoema VSCode extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-04-03

### Added
- Initial release of Synoema VSCode extension
- Syntax highlighting for `.sno` files with full token support
- Language configuration for brackets, comments, and indentation
- Run command (`Ctrl+Shift+R` / `Cmd+Shift+R`) — execute Synoema programs
- JIT command (`Ctrl+Shift+J` / `Cmd+Shift+J`) — compile to native code
- Eval command — evaluate selected expressions
- Integrated output panel for results and error display
- Configuration options for CLI path, output behavior, and timeout
- Comprehensive README with usage examples and troubleshooting
- User-friendly error messages and CLI availability checking
